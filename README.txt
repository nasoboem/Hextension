Hextension

Voraussetzung:
Installation von Go Version 1.11.2 oder neuer.
Für Windows muss das Paket gfx2 von Stefan Schmidt vom 3.3.2018 installiert werden. 

Hextension gibt es in zwei Versionen
    • Linux
        ◦ basierend auf dem gfx-Paket (beigefügt)
    • Windows
        ◦ basierend auf dem gfx2-Paket (nicht beigefügt)

In beiden Fällen liegen die ausführbaren Dateien im Ordner Spiel.

Hextension → Linux o. Windows → Spiel

Ausführbare Dateien:
    • Hextension.go → das Spiel
    • bestgame-test.go → berechnet die optimale Position für ein zufällig gewähltes Spiel (nicht wirklich, aber man kann sich den Anfang ansehen ~ ersten 10.000.000 Versuche, wenn man geduldig ist)
